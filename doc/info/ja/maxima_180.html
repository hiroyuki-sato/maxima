44.2 Functions and Variables for contrib_ode

関数: contrib_ode (eqn, y, x)

独立変数xと従属変数yに関するODE eqnの解のリストを返します。

Package contrib_ode ·
関数: odelin (eqn, y, x)

odelinは 独立変数xと従属変数yに関する 一階および二階線形斉次ODEを解きます。 ODEの基本的な解一式を返します。

二階ODEに対して、odelinは、 与えられた特殊関数を使って解を探索する BronsteinとLafailleによる方法 を使います。 

(%i1) load("contrib_ode");

(%i2) odelin(x*(x+1)*'diff(y,x,2)+(x+5)*'diff(y,x,1)+(-4)*y,y,x);
...trying factor method
...solving 7 equations in 4 variables
...trying the Bessel solver
...solving 1 equations in 2 variables
...trying the F01 solver
...solving 1 equations in 3 variables
...trying the spherodial wave solver
...solving 1 equations in 4 variables
...trying the square root Bessel solver
...solving 1 equations in 2 variables
...trying the 2F1 solver
...solving 9 equations in 5 variables
       gauss_a(- 6, - 2, - 3, - x)  gauss_b(- 6, - 2, - 3, - x)

Package contrib_ode ·
関数: gauss_a (a, b, c, x)

gauss_a(a,b,c,x)とgauss_b(a,b,c,x)は2F1幾何関数です。 それらは超幾何微分方程式 x(1-x) diff(y,x,2) + [c-(a+b+1)x diff(y,x) - aby = 0 の任意の２つの独立解を表します(A&S 15.5.1)。

これらの関数は odelinとcontrib_odeが返すODEの解の中でだけ使われます。 これらの関数の定義と使用はMaimaの将来のリリースで変わるかもしれません。

gauss_b, dgauss_a, gauss_bも参照してください。

Package contrib_ode ·
関数: gauss_b (a, b, c, x)

See gauss_a.

Package contrib_ode ·

Package contrib_ode ·
